# Air-Drums

Created as a project for *Intoduction To Engg. Design* (Semester 2, IIIT-D).

## Uses:
* NodeMCU
* Gyroscope
* Piezoelectric Sensor
* Speakers

The `.ino` file holds all the code for NodeMCU to work.
